## ğŸ“ QUICK CONCEPTS

### Components
Reusable blocks of code. Use them anywhere, anytime.

```jsx
<ListItems items={items} />  // One line does all the work!
```

### Props
Data passed to components to customize them.

```jsx
<Card title="My Title" text="My text" icon="icon.svg" />
```

### Pages
Combine components to build entire pages.

```jsx
<HeroSection /> + <ListItems /> + <CardGrid /> = Full page!
```
